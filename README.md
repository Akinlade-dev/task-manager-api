# Task Manager API

RESTful API for managing tasks with CRUD operations.

## Tech Stack
- Node.js
- Express
- Render

## Live Demo
[https://task-manager-api-bt1u.onrender.com](https://task-manager-api-bt1u.onrender.com)

## API Endpoints
- `GET /tasks` - Get all tasks
- `GET /tasks/:id` - Get single task
- `POST /tasks` - Create new task
- `PUT /tasks/:id` - Update task
- `DELETE /tasks/:id` - Delete task

## Features
- In-memory data storage
- JSON request/response
- Error handling
- RESTful design

## How to Run
1. Clone: `git clone https://github.com/akinlade-dev/task-manager-api.git`
2. Install: `npm install`
3. Start: `npm start`
4. Test: Visit `http://localhost:3000/tasks`
